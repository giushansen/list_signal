Open each website link, check if Business Model, Industry, and Revenue look correct, then mark each one. When done, click "Save for Claude" to export a JSON file to share.

How to use this page

Columns explained:

  • BM = Business Model (Ecommerce, SaaS, Agency, Consulting, Media, etc.)
  • Ind = Industry (Fintech, Healthcare, Fashion, DevTools, etc.)
  • Rev = Revenue estimate bracket (<$1M, $1M-$10M, $10M-$100M, $100M-$1B, $1B+)
  • Evidence = All signals that contributed to the revenue estimate, color-coded by bracket they point to: micro small mid_market enterprise large_ent
  • Issues = Auto-detected red flags (shown as yellow text). E.g. "Wix-hosted = IL flag is server location, not business", "default Shopify password page", "no traffic rank but revenue > $1M"

Workflow: Click the domain link to open the website. Glance at the site. Does the BM/Ind/Rev feel right? Click checkmark or X for each. Use the Notes field to say what the correct value should be (e.g. "should be Agency" or "looks like $10M+"). When you're done reviewing, hit "Save for Claude" - it downloads a JSON with all your verdicts + the raw data so Claude can analyze patterns and fix the code.
